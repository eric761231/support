# 天堂經典版體力監控器 (Lineage Classic HP Monitor)

這是一個專為《天堂經典版》設計的輔助工具，透過光學字元辨識 (OCR) 技術即時監測角色的血量 (HP) 與法力 (MP)，並在低於設定門檻時自動發送鍵盤指令（如使用補血藥水）。

## 核心功能

*   **即時監測**：支援 HP 與 MP 同時監測。
*   **自動補給**：自訂補血/補魔百分比門檻，低於門檻時自動按下指定熱鍵 (F5-F12)。
*   **OCR 技術**：採用 Tesseract OCR 與 OpenCV 影像處理，精準讀取遊戲畫面數值。
*   **設定保存**：自動記憶您的監測門檻、視窗標題與熱鍵設定。
*   **視窗選擇**：自動列出執行中的遊戲視窗，不需手動輸入標題。
*   **深色模式 UI**：簡潔實用的 WPF 介面，支援高對比與深色主題。

## 系統需求

*   **作業系統**：Windows 10/11
*   **.NET Runtime**：.NET Framework 4.7.2 或更高版本
*   **遊戲視窗**：需以「視窗化」模式執行遊戲。

## 安裝與執行

1.  下載此專案並編譯（使用 Visual Studio 2022+）。
2.  確保 `tessdata` 資料夾包含 `eng.traineddata` 與 `chi_tra.traineddata`。
3.  以「系統管理員身分」執行 `LinHPMonitor.exe` (為了能發送指令到遊戲視窗)。

## 使用指南

1.  **選擇視窗**：從下拉選單選擇您的遊戲視窗。
2.  **設定門檻**：在「補血點(%)」與「補魔點(%)」輸入您想要的觸發值。
3.  **設定熱鍵**：選擇對應遊戲中快捷列的按鍵。
4.  **開始監測**：點擊「開始監測」按鈕即可。

## 注意事項

*   本工具僅供學術與技術研究參考，請勿用於非法用途。
*   使用前請確認遊戲環境是否允許辅助工具，避免帳號風險。

## 技術架構

*   **語言**：C# / WPF
*   **影像處理**：OpenCvSharp
*   **文字辨識**：Tesseract OCR
*   **輸入控制**：Win32 API (SendInput)
